# GoatGuard — imágenes y videos

Capturas y clips demo del proyecto **GoatGuard** (network monitoring + 2FA).

## Qué subir
- **Imágenes**: PNG / JPG / WebP. Capturas de la app, dashboards, flujos 2FA.
- **Videos**: MP4 (H.264). Clips cortos de demo.

## Recomendaciones
- **Nombres descriptivos**: `goatguard-dashboard.png`, `goatguard-2fa-flow.png`, `goatguard-demo.mp4`.
- **Tamaño de imágenes**: ancho 1280–1920 px suele bastar; comprime para que pesen poco.
- **Videos**: mantenlos **cortos y comprimidos** (idealmente < 5 MB). El repo y Vercel no son ideales para vídeos pesados; si tienes un clip largo, súbelo a **YouTube/Cloudinary** y lo enlazamos en lugar de versionarlo.
  - Para un MP4 que reproduzca bien en web, usa un **póster** (imagen de portada) — me lo dices y lo configuro.
